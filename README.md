### Installation
1) GitHub repository  
```
 https://github.com/vikram17dp/ExpenseTracker.git
```
2) Install dependencies  
```
cd Expense-Tracker_Mern  
cd frontend
npm install
cd..
cd client
npm install
```

3) Run the server
```
npm run server
```
4) Start a new terminal and run react's development server
```
cd social-media-app
cd frontend
npm start
```
